# 🍕 Ak Pizza 

A modern, fast, and responsive pizza ordering application built with **React**, **Redux Toolkit**, and **Tailwind CSS**.

## 🚀 Features

- **Menu Integration:** Fetches real-time data from a custom API.
- **Cart Management:** Add, remove, and update pizza quantities using Redux state.
- **User Profiles:** Persistent user data across the ordering process.
- **Geolocation:** Automatic address lookup using the browser's Geolocation API.
- **Responsive Design:** Mobile-first UI styled with Tailwind CSS.

## 🛠️ Tech Stack

- **Framework:** React (Vite)
- **Routing:** React Router (using modern Data APIs)
- **State Management:** Redux Toolkit
- **Styling:** Tailwind CSS

## 📦 Installation & Setup

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to start the development server
